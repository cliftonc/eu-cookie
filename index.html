Simple EU Cookie Banner

Examples

  • Default values, bottom of page and fixed.
  • At top of page, relative (with push down), with automatic acceptance after 15 seconds.
  • At top of page, fixed, frequency capped (try refreshing more than 3 times).

Availalbe configuration parameters

You can configure the script by setting options into the parameter 'window.EUCMOptions' as such:


  window.EUCMOptions = {
    elementId: '',
    acceptTimeout: 15,    
    frequencyCap: 3,
    policyUrl:'/privacypolicy',
    cookie: {
      key: 'eu-cookie-acceptance',
      expires: 30,
      path: '/',
      domain: '',
      secureOnly: false
    },
    copy: {
      notice:'This site uses cookies.  By continuing to browse the site you are agreeing to our use of cookies:',
      accept:'Accept and hide this notice',
      acceptTooltip:'Click here to accept this and close this notice ...',
      info:'Find out more information here',
      infoTooltip:'Click here to find out more information about how we use cookies ...',
      close:'X',
      closeTooltip:'Click here to accept this and close this notice'
    },
    color: {
      background:'#3F3F3F',      
      text:'#DFDFDF',
      link:'#FFFFFF',
      linkHover:'#EFEFAF'
    },
    topOrBottom:'bottom',
    position:'fixed'    
  }

The values above are the defaults. You only need to specify the values that you would like to over-ride in the defaults (see the examples).
