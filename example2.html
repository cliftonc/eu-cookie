Eu-cookie

At top of page, relative (with push down), notice changed, acceptance link removed, different color, with automatic acceptance after 15 seconds (wait for it!).

You an change any of the copy as per the configuration shown in the index page.

Example 2

<script>
    window.EUCMOptions = {      
        acceptTimeout: 15,    
        policyUrl: 'policy.html',
        color: {
          background: '#549898'
        },
        cookie: {
          key:'EUC-EXAMPLE2'
        },
        copy: {
          notice:'This is a custom notice, say whatever you like!',
          accept:''
        },
        topOrBottom:'top',
        position:'relative'
    }
</script>
<script type="text/javascript" async="" src="path/to/eu-cookie.min.js"></script>

Clearly, if you accept, you need to delete your cookies for this site for it to continue to appear again!
